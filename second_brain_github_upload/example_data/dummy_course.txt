Lezione 1: I Database Vettoriali.
Un database vettoriale (come ChromaDB) memorizza le informazioni sotto forma di embedding, ovvero vettori matematici ad alta dimensionalità. Questo permette di effettuare ricerche per similarità semantica, anziché per semplice corrispondenza esatta delle parole chiave.
